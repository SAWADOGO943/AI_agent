# Frontend Chatbot IA + Agent

Interface web moderne et simple pour interagir avec le backend FastAPI.

## Fonctionnalités

- **Design moderne** : Interface avec dégradé, ombres et animations subtiles
- **Icônes FontAwesome** : Indicateurs visuels pour chaque section
- **Animations de chargement** : Indicateurs de progression pendant les requêtes
- **Responsive** : Adapté aux mobiles et tablettes
- **Chat Simple** : Discussion basique avec Gemini AI
- **Analyse Agent** : Analyse de texte avec planification, exécution d'outils et synthèse
- **Status** : Vérification de l'état du serveur et de l'agent

## Technologies

- HTML5
- CSS3 (avec animations et responsive design)
- JavaScript (ES6+)
- Fetch API pour les requêtes HTTP
- FontAwesome pour les icônes

## Utilisation

1. Assurez-vous que le backend est en cours d'exécution sur `http://localhost:8000`
2. Ouvrez `index.html` dans un navigateur web moderne
3. Utilisez les sections pour interagir avec l'API

## Notes

- Nécessite une connexion internet pour charger FontAwesome
- Compatible avec les navigateurs modernes (Chrome, Firefox, Safari, Edge)
